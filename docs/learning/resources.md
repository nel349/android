## Reference Apps (Study These!)

**Production-Quality Codebases:**

1. **[Nowinandroid](https://github.com/android/nowinandroid)** (Google)
   - Multi-module Clean Architecture
   - 100% Jetpack Compose
   - MVI pattern, Convention plugins
   - **Study for:** Architecture, modularization, Compose best practices

2. **[Tivi](https://github.com/chrisbanes/tivi)** (Chris Banes)
   - TV show tracking app
   - Multi-module, Compose, Hilt
   - **Study for:** Real-world Compose, navigation, data layer

3. **[Plaid](https://github.com/nickbutcher/plaid)** (Nick Butcher)
   - Material Design showcase
   - Custom animations, transitions
   - **Study for:** UI polish, animations

4. **[Iosched](https://github.com/google/iosched)** (Google I/O app)
   - Large-scale production app
   - **Study for:** Complex features, scalability

5. **[Compose Samples](https://github.com/android/compose-samples)** (Google)
   - Official Compose sample apps
   - **Study for:** Compose patterns, state management

## How to Use These Resources

**Month 1-2 (Foundation):**
- Study: Kotlin docs, Android Codelabs, reference apps (Nowinandroid, Tivi)
- Practice: LeetCode easy problems, build Weather app
- Community: Join r/androiddev, Android Dev Discord

**Month 3-4 (Advanced + Interview Prep):**
- Study: Nowinandroid architecture, "Effective Kotlin", "Cracking the Coding Interview"
- Practice: LeetCode medium/hard, mock interviews (Pramp 2x/week)
- Community: Follow Twitter devs, engage in discussions

**Month 5-6 (Active Interviewing):**
- Study: Company-specific LeetCode problems, review STAR stories
- Practice: Mock interviews 3x/week (Interviewing.io), timed LeetCode
- Network: LinkedIn outreach, referrals, Blind discussions

---

**Remember:** Quality > Quantity. Master one resource deeply rather than skimming many. Your goal is FAANG-level expertise, not completion of every resource listed.
